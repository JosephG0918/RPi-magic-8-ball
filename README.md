# Magic Eight Ball Raspberry Pi Project

A handheld, interactive Magic Eight Ball built with a Raspberry Pi Zero W, a motion sensor, and a small OLED display. Shake it to get classic eight-ball answers!  

Full hardware, software, and 3D-printing instructions are in the [docs folder](./docs/).
